# Logseq

**Category:** Note-taking & Workspaces
**Type:** Local-first Software
**Jurisdiction:** Global (Open Source / Self-Hosted Exception)
**Website:** [logseq.com](https://logseq.com/)

## What It Is
Logseq is a privacy-first, open-source knowledge base that works on top of local plain-text Markdown and Org-mode files.

## Hosting Options
Local-first. Files are stored directly on your local machine. Can be synced using an EU-hosted service like Nextcloud or Tresorit.

## Compliance Notes
- **Self-Hosting / Local Exception:** Data never leaves your device unless explicitly synced using a service you control.
- No cloud vendor lock-in.
