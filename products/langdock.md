# Langdock

**Category:** AI Assistant, Enterprise AI Platform
**Type:** Managed SaaS / Own Cloud / On-Prem
**Jurisdiction:** Germany (EU)
**Website:** [langdock.com](https://langdock.com)

## What It Is

Langdock is a Berlin-based enterprise AI platform that consolidates AI chat, workflow automation, and internal knowledge search in a single GDPR-compliant product. It decouples the UI from model providers, letting teams use multiple LLMs (Mistral, Claude, GPT) through one interface with EU-only data processing.

## Hosting Options

| Option | Description |
|---|---|
| **Multi-tenant SaaS** | Hosted in Europe |
| **Single-tenant / Dedicated** | Managed dedicated deployment for larger customers |
| **Own Cloud** | Dedicated deployment in your own cloud |
| **On-Prem** | Kubernetes-based on-premise deployment for large enterprises |
