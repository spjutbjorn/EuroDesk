# AppFlowy

**Category:** Note-taking & Workspaces
**Type:** Self-hosted (Exception)
**Jurisdiction:** US/Global (Open Source / Self-Hosted Exception)
**Website:** [appflowy.io](https://appflowy.io/)

## What It Is
AppFlowy is an open-source alternative to Notion, offering workspaces, wikis, and task management.

## Hosting Options
Must be self-hosted on EU-owned infrastructure to satisfy EuroDesk requirements. It also offers a local-first offline mode.

## Plans (2026 Estimates)

| Plan | Price (User/Mo) | Key Features |
|---|---|---|
| **Free** | $0 | 1 user seat, 3 guest editors |
| **Teams** | ~$10 | Collaborative self-hosting, AI access |
| **Enterprise**| Custom | SAML SSO, advanced admin |

> **Note:** AppFlowy has transitioned to an open-core model. Self-hosting for more than 2-3 collaborators requires a Team plan.

## Key Features
- Database views (Kanban, Grid, Calendar).
- Local-first data storage.
- Collaborative workspaces.
- Plugin system for custom functionality.

## Compliance Notes
- **Self-Hosting Exception:** Included because it is open-source and allows for complete self-hosting.
- As a local-first application, it provides excellent privacy by keeping data on your device or your self-hosted EU server.
