# Mullvad Browser

**Category:** Web Browser
**Type:** Local Software
**Jurisdiction:** EU (Sweden)
**Website:** [mullvad.net/en/browser](https://mullvad.net/en/browser/)

## What It Is

A privacy-focused web browser developed in collaboration between Mullvad VPN and the Tor Project. It is designed to minimize tracking and fingerprinting without routing traffic through the Tor network.

## Why it fits EuroDesk

- **Jurisdiction:** Mullvad is based in Sweden.
- **Anti-fingerprinting:** Creates a similar fingerprint for all its users, making it difficult to track individuals across the web.
- **Telemetry:** Completely removes all telemetry found in standard browsers like Chrome or Edge.

## Compliance Notes

- **Jurisdiction:** Developed by Mullvad VPN, based in Sweden (EU).
- **Anti-telemetry:** Completely removes all telemetry found in standard browsers like Chrome or Edge.
- **Privacy:** As a local software product, it does not store data in the cloud. All browsing activity is local and transient by default.
- **GDPR:** Does not collect or transmit personal data to the browser developer.
