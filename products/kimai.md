# Kimai

**Category:** Time Tracking & Invoicing
**Type:** Self-hosted / SaaS
**Jurisdiction:** EU (Germany)
**Website:** [kimai.org](https://www.kimai.org/)

## What It Is
Kimai is a free, open-source time-tracking application that supports invoicing, multi-user environments, and comprehensive reporting.

## Hosting Options
Can be fully self-hosted on EU infrastructure or used as a SaaS hosted in Germany.

## Plans (2026)

| Option | Price (Month) | Key Features |
|---|---|---|
| **Self-hosted** | $0 | Unlimited users, open-source |
| **Kimai Cloud** | ~€9 | Managed, backups, easy setup |

> **Note:** Kimai is one of the most popular open-source time tracking tools in Europe, compliant with the EU "Working Time Directive" requirements for recording daily working time.

## Key Features
- Multi-user and team management.
- Detailed reporting and data export.
- Automated invoicing based on tracked time.
- REST API for integrations.

## Compliance Notes
- Open-source and developed in Germany.
- No vendor lock-in; you own your data.
- GDPR compliant.
