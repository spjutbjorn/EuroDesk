# AnythingLLM

**Category:** Private AI / Knowledge Base
**Type:** Self-hosted / Desktop App
**Jurisdiction:** EU-hostable (Open Source)
**Website:** [useanything.com](https://useanything.com)

## What It Is

AnythingLLM is an all-in-one platform that turns your internal documents, PDFs, and links into a private searchable knowledge base that you can chat with using Large Language Models. It excels at local RAG (Retrieval-Augmented Generation) and is designed for privacy and ease of use.

## Good Fit For

- Building a private "company brain" from internal documentation
- Legal and research teams needing to query hundreds of documents securely
- Individuals wanting a local AI assistant on their workstation

## Setup Options

| Option | Description |
|---|---|
| **Desktop App** | Runs locally on Windows, macOS, or Linux. Models can be built-in or external. |
| **Docker (Self-hosted)** | Run as a team-wide server on any EU VPS. |

## Key Features

- **Document Ingestion:** Drag and drop PDFs, text, or web links to build "Workspaces".
- **Built-in LLM Engine:** Can run models locally without any external API.
- **Multi-user Support:** Create workspaces for different teams with separate permissions.
- **Embedded Search:** Uses a local vector database to find relevant document sections.
- **Customizable:** Choose your embedding engine and vector store.

## Compliance Notes

- **Data Sovereignty:** All document processing and storage happen on your local machine or server.
- **Zero Cloud Exposure:** When configured with local models (e.g., via EULLM or built-in engine), no data ever leaves your network.
- **GDPR:** Ideal for processing sensitive personal data that cannot be sent to US-based AI providers.

## EuroDesk Verdict

AnythingLLM is the most practical tool for turning a collection of files into an actionable, sovereign AI knowledge base.
