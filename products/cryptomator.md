# Cryptomator

**Category:** Encryption / Data Protection
**Type:** Local Software
**Jurisdiction:** Germany (EU)
**Website:** [cryptomator.org](https://cryptomator.org)

## What It Is

Cryptomator is an open-source file encryption tool for protecting data at rest before it is uploaded to cloud storage or synchronized across devices. It creates encrypted vaults that can be stored inside Nextcloud, Tresorit, external drives, or local folders while keeping filenames and contents protected.

## Hosting Options

| Option | Description |
|---|---|
| **Desktop app** | Runs locally on user devices for encrypted vault access |
| **Mobile app** | Opens encrypted vaults on mobile devices |
| **Cloud storage companion** | Store encrypted vaults inside EU-hosted storage such as Nextcloud or Tresorit |

## Good Fit For

- Protecting highly sensitive files before sync or backup
- Legal, HR, finance, and compliance document handling
- Laptop-local encryption beyond default folder storage
- Shared encrypted archives with controlled access

## Key Features

- Client-side encryption before upload
- Open-source vault format
- Works with common cloud and local storage providers
- File and filename encryption
- Available on both Windows and macOS, plus Linux and mobile platforms

## Compliance Notes

- Cryptomator protects data in storage, but access management still depends on the surrounding platform.
- For strongest sovereignty, combine it with EU-hosted storage and EU-controlled backup targets.
- Shared vault workflows require careful key and password handling.

## Trade-offs

- Less convenient than plain file sync for collaborative editing
- Not a replacement for full disk encryption or identity controls
- Shared editing inside encrypted vaults can be awkward for large teams

## EuroDesk Verdict

Cryptomator fills an important gap in EuroDesk by adding practical client-side encryption for sensitive files, so data can remain protected even inside synced or archived storage.
