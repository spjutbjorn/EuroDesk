# Mistral AI / Le Chat

**Category:** AI Assistant, LLM
**Type:** Managed SaaS / Self-hosted / API
**Jurisdiction:** France (EU)
**Website:** [mistral.ai](https://mistral.ai)

## What It Is

Mistral AI is Europe's leading LLM provider, based in Paris. Le Chat is the consumer and enterprise AI assistant — comparable to ChatGPT/Claude but with full EU jurisdiction. Mistral also provides open-weight models for self-hosting with zero API costs.

## Hosting Options

| Option | Description |
|---|---|
| **Le Chat (managed)** | Mistral-hosted SaaS, EU data centers |
| **Le Chat Enterprise** | Self-hosted, private cloud, or Mistral-hosted — your choice |
| **Self-hosted open-weight** | Run Mistral models on any EU infrastructure (Hetzner, OVH, Scaleway) |
| **API** | Pay-per-token, EU-hosted endpoints |

## Key Features

- **Le Chat:** Web-based assistant with chat, search, document analysis, and image generation.
- **Mistral Vibe:** Integrated development environment and CLI for real-time coding.
- **Deep Research:** Advanced high-reasoning mode for complex technical reports.
- **Canvas:** Collaborative editing for code and documents.
- **Enterprise Features:** SSO (SAML), admin console, audit logs, and on-prem/private cloud deployment.
