# SwissTransfer

**Category:** File Transfer
**Type:** SaaS
**Jurisdiction:** Switzerland (Adequacy-Approved)
**Website:** [swisstransfer.com](https://www.swisstransfer.com/)

## What It Is
SwissTransfer is a free, secure file transfer service based in Switzerland. It allows sending large files (up to 50 GB per transfer) without an account, making it a sovereign European alternative to WeTransfer.

## Hosting Options
SaaS. Hosted exclusively in Infomaniak's ecologically responsible datacenters in Switzerland.

## Key Features
- **Free usage:** No subscription or account required.
- **Large limits:** Up to 50 GB per transfer.
- **Privacy:** End-to-end encrypted transfer and Swiss data residency.
- **Customizable:** Set expiration dates (up to 30 days) and password protection.

## Compliance Notes
- Switzerland has an Adequacy Decision from the European Commission, providing equivalent protection to the GDPR.
- Files are encrypted in transit and at rest.
- No registration required.
