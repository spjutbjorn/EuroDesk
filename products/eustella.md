# eustella

**Category:** AI Assistant
**Type:** Managed SaaS (mobile-first)
**Jurisdiction:** Austria (EU)
**Website:** [eustella.com](https://eustella.com)

## What It Is

eustella is a Vienna-based AI assistant built as a European alternative to ChatGPT. It runs exclusively on European and open-source AI models — no US Big Tech models. Designed mobile-first for smartphones and wearables, with agentic capabilities that go beyond simple chat.

## Hosting Options

| Option | Description |
|---|---|
| **Managed SaaS** | EU data centers, no self-hosted option announced |

## Key Features

- Chat, document analysis, code generation, research
- Agentic AI: manages calendar, curates playlists, delivers news briefings, analyzes finances
- 8 European languages: EN, DE, FR, ES, IT, NL, PT, PL
- Community features planned: share agents with groups/teams
- No US models used — only European and open-source LLMs
