# DeepL

**Category:** Translation
**Type:** Managed SaaS / API
**Jurisdiction:** Germany (EU)
**Website:** [deepl.com](https://www.deepl.com/)

## What It Is

DeepL is an AI translation platform known for high-quality translations, especially across European languages. It offers both a web product and an API for integrating translation into internal workflows.

## Hosting Options

| Option | Description |
|---|---|
| **DeepL Pro** | Managed EU-hosted translation and API service |
| **API integration** | Connect internal tools to DeepL Pro with controlled data flows |

## Why It Fits EuroDesk

- **Jurisdiction:** DeepL SE is based in Cologne, Germany.
- **Enterprise privacy:** DeepL Pro is designed for business use with stronger confidentiality expectations.
- **Quality:** Often outperforms major US-based translation tools for European languages.

## Compliance Notes

- Use `DeepL Pro` rather than free consumer usage for business-sensitive text.
- Route only the content that actually needs translation, and avoid unnecessary personal data.
- Review retention, logging, and API-key handling in any internal integrations.
