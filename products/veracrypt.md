# VeraCrypt

**Category:** Encryption / Data Protection
**Type:** Local Software
**Jurisdiction:** France (EU)
**Website:** [veracrypt.fr](https://veracrypt.fr)

## What It Is

VeraCrypt is an open-source disk and container encryption tool for creating encrypted volumes, securing removable drives, and protecting sensitive data on laptops and workstations. It is useful when teams need stronger local protection for files, archives, or portable storage beyond plain synced folders.

## Hosting Options

| Option | Description |
|---|---|
| **Desktop app** | Runs locally on user devices for encrypted containers and volumes |
| **Portable drive protection** | Encrypts USB drives and external media for offline or transport use |

## Good Fit For

- Encrypting portable drives and offline archives
- Protecting sensitive project folders on laptops
- Securing exported records, legal bundles, or investigation material
- Creating encrypted containers for handover or long-term storage

## Key Features

- Encrypted file containers
- Full volume and removable media encryption
- Cross-platform support on Windows and macOS, plus Linux
- Open-source cryptographic implementation
- Strong fit for offline and endpoint-local protection

## Compliance Notes

- VeraCrypt protects local data at rest, but does not replace identity, backup, or document access controls.
- Operational use depends on strong passphrase management and documented recovery procedures.
- For collaborative storage, pair it with governed file-sharing and key-handling processes.

## Trade-offs

- Less seamless than regular file-sync collaboration
- Not ideal for many concurrent editors on the same encrypted container
- Requires user discipline around mounting, unmounting, and password recovery

## EuroDesk Verdict

VeraCrypt complements Cryptomator by covering encrypted containers, removable media, and endpoint-local storage protection for highly sensitive data.
