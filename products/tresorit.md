# Tresorit

**Category:** File Transfer & Storage

**Jurisdiction:** Switzerland (Adequacy-Approved)

## What It Is
Tresorit is an end-to-end encrypted file sync and sharing solution designed to safeguard confidential information.

## Hosting Options
SaaS. Data is stored in EU/Swiss datacenters.

> **Jurisdiction:** Based in Switzerland, ensuring strong privacy protections and zero CLOUD Act exposure.

## Key Features
- Zero-knowledge end-to-end encryption.
- Secure file sharing and collaboration.
- Administrative controls and user management.
- Digital signatures and tracking.
